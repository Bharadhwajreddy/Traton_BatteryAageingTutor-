# Literature Archive — BatteryAgeTutor

This folder contains structured reference materials for key battery-ageing publications
that are discussed in the app. No copyrighted text is reproduced — every document here
is an original summary/analysis written for educational use, plus open bibliographic data.

Each file is named `<firstauthor><year>.md`.

## Contents

| File | Paper | Why it matters |
|---|---|---|
| `schmalstieg2014.md` | Schmalstieg et al., J. Power Sources 257 (2014) | Defining semi-empirical NMC holistic calendar+cycle model from ISEA/RWTH; 549+ citations |
| `ecker2014.md` | Ecker et al., J. Power Sources 248 (2014) | Companion experimental dataset and ageing-factor study that underpins Schmalstieg 2014 |
| `references.bib` | BibTeX bibliography | Machine-readable citations for all covered papers |

## Reading guide

Each `.md` file follows this structure:
1. Bibliographic details and open-access links
2. Why this paper matters (context in the literature)
3. Experimental design (what was tested, at what conditions)
4. Model structure (equations written in our own words, not copied)
5. Key findings
6. Limitations and what later work addressed
7. Connections to the tutor app's interactive tools
